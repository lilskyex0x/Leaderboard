## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need: a properly working pc or laptop and a modern web browser. And also you must install node.js in your computer.

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/lilskyex0x/Leaderboard.git
```

### Install

Install this project with:

```sh
  cd my-folder
  git clone https://github.com/lilskyex0x/Leaderboard.git
```

### Usage

To run the project, you will have to install node.js first then type ```npm run start``` inside your terminal.

### Run tests

you will run automatically if you have already install webpack. My document has already installed webpack so if you clone my repo it should be fine.

### Deployment
